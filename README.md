# NJ Pro Cleaning Services Website (Vercel-ready)

## 1) Put your logo + photos
- Replace `public/logo.svg` with your real logo (or set the code to use `logo.png`)
- Put your job photos into `public/work/` using these filenames:
  - gym-weights-1.jpg
  - gym-lounge-1.jpg
  - office-tiles-1.jpg
  - venue-carpet-1.jpg
  - venue-floor-1.jpg
  - restroom-black-tiles-1.jpg
  - restroom-sinks-1.jpg
  - office-reception-1.jpg
  - window-clean-1.jpg
  - window-clean-2.jpg
  - gym-cardio-1.jpg
  - office-carpet-1.jpg
  - office-carpet-2.jpg

## 2) Run locally (optional)
npm install
npm run dev

## 3) Deploy on Vercel
Push this project to GitHub, then import repo in Vercel.

## 4) Important
Update `SITE.url` in src/App.jsx to your real deployed domain for SEO.
